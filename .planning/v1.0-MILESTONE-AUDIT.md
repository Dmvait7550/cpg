- Affected: CONN-05 (warning logging works, session summary counter does not)

- Location: `pkg/k8s/cluster_dedup.go:39`, `pkg/hubble/pipeline.go:111`
- Affected: DEDP-02 (single-namespace works, multi-namespace edge case)
