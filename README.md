## 🚀 What cpg Does

cpg helps you set network policies using data from hubble relay. Network policies control how devices and applications communicate. Managing these policies can improve security and performance.

- Generate new policies automatically based on your network activity
- Edit policies easily using the built-in interface
- Apply policies to networks running Cillium, a cloud-native networking tool
- Monitor real-time data from hubble relay to adjust policies as needed

## ⚙️ Basic Usage Guide

Once you open cpg, you will see a simple interface with three main sections:

1. **Policy Generator**  
   Use this section to create new network policies automatically. Click "Generate" to start. The program connects with hubble relay to gather current network data.

2. **Policy Editor**  
   Edit existing policies here. Select a policy from the list and make changes using clear options. Save the policy before applying.

3. **Apply Policies**  
   After generating or editing a policy, click "Apply" to activate it on your network.

The interface uses straightforward buttons and menus. You do not need to write any code or scripts.
